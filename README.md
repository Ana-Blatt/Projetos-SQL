Este repositório reúne projetos práticos desenvolvidos durante meus estudos em SQL, com foco em criação e manipulação de bancos de dados, estruturação de tabelas e organização de informações.
Os projetos simulam situações comuns de sistemas utilizados por empresas, como catálogo de produtos, gestão de contatos, controle de estoque e organização de tarefas.

PROJETOS:

#Catálogo de Fimes - Estrutura banco de dados para armazenar informações de contatos:
- id = Identificador único do filme;
- título = Nome do filme;
- diretor = Diretor respos[avel pelo filme;
- ano_publicacao = Ano de lançamento;
- geneto = Gênero do filme.
  
Objetivo: Simular um sistema simples de cadastro de contatos.

#Gestão de Estoque - Banco de dados voltado para controle básico de estoque de produtos:
- id = identificados do produto;
- nome_produto = Nome do produto;
- quantidade = Quantidade disponível em estoque;
- preco_compra = Valor de compra do produto;
- preco_venda = valor de venda do produto.

Objetivo: Simular o controle de produtos e preços dentro de um estoque.
